# Medical Report AI Analyzer

A full-stack AI-powered web application that helps users understand medical lab reports by analyzing PDFs using AI.

## Features

- User authentication (Supabase)
- Upload medical lab report PDFs
- AI-based report explanation using Google Gemini
- Secure user data with Row Level Security

## Tech Stack

- Frontend: Next.js, Tailwind CSS
- Backend: Supabase, PostgreSQL
- AI: Google Gemini API

## How it works

1. User logs in
2. Uploads a lab report PDF
3. Text is extracted from the PDF
4. AI analyzes the report
5. Explanation is shown to the user
